## Durée des signaux

### Unités temporelles
- **Point (·)** : 1 unité
- **Trait (-)** : 3 unités
- **Espace entre éléments** : 1 unité
- **Espace entre lettres** : 3 unités
- **Espace entre mots** : 7 unités

### Vitesse de transmission
- **Débutant** : 5-10 mots/minute (WPM)
- **Opérateur expérimenté** : 15-25 WPM
- **Record** : 75 WPM (Ted McElroy, 1939)

## Adaptation pour le MML

### Caractères prioritaires
Le MML privilégie les caractères Morse courts pour les balises fréquentes :
- **T** : - (1 trait) → Très rapide
- **E** : · (1 point) → Ultra rapide
- **H** : ···· (4 points) → Rapide malgré la longueur
- **M** : -- (2 traits) → Efficace

### Optimisations MMLC
La version compressée MMLC utilise les chiffres (0-9) qui ont des codes Morse courts :
- **1** : ·---- (1+4 = 5 unités)
- **2** : ··--- (2+3 = 5 unités)
- **3** : ···-- (3+2 = 5 unités)
- **4** : ····- (4+1 = 5 unités)
- **5** : ····· (5 unités)

### Transmission pratique
Pour une transmission MML optimale :
1. Utiliser les prosignes pour structurer la communication
2. Grouper les éléments similaires
3. Répéter les informations critiques
4. Utiliser les codes abrégés pour les termes fréquents
